# Windows Management Framework (WMF) 5.1 版本資訊 (預覽) #

> 注意：本資訊尚屬初始版本，後續有可能變更。

本節中的文件適用於即將推出的 WMF 5.1 Preview。 WMF 5.1 Preview 的發行日期尚未決定。 此為初稿內容，後續數週會經常更新。
