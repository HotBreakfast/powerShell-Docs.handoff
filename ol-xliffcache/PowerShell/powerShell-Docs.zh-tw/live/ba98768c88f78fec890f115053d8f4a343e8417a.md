# 適用於 Linux 的內建預期狀態設定資源

資源是可讓您撰寫 PowerShell 預期狀態設定 (DSC) 指令碼的建置組塊。 DSC for Linux 隨附一組內建功能，可用於設定資源，例如檔案和資料夾、套件、環境變數，以及服務和處理序。

## 內建資源
