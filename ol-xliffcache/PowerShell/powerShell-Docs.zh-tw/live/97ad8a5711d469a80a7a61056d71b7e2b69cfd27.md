# MSFT_DSCLocalConfigurationManager 類別的 DisableDebugConfiguration 方法

停用 DSC 資源偵錯。

語法
------

```mof
uint32 DisableDebugConfiguration();
```

參數
----------

這個方法沒有任何參數。

## 傳回值
------------

若成功即傳回零；否則傳回錯誤碼。

## 備註

此為靜態方法。

## 需求
------------
>**MOF：**DscCore.mof

>**Namespace**：Root\Microsoft\Windows\DesiredStateConfiguration
