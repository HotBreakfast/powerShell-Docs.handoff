# Usando o Windows PowerShell
O [!INCLUDE[wps_1](../Token/wps_1_md.md)] é um shell de linha de comando baseado em tarefas e uma linguagem de script cujo design está especialmente voltado para a administração do sistema. Baseado no Microsoft .NET Framework, o [!INCLUDE[wps_2](../Token/wps_2_md.md)] ajuda profissionais de TI e usuários avançados a controlar e automatizar a administração de sistemas operacionais Windows e de aplicativos que são executados no Windows.

Os recursos desta seção ajudarão você a aprender mais sobre o [!INCLUDE[wps_2](../Token/wps_2_md.md)], os recursos incluídos no [!INCLUDE[wps_2](../Token/wps_2_md.md)] e o editor gráfico do [!INCLUDE[wps_2](../Token/wps_2_md.md)], o Ambiente de Script Integrado do [!INCLUDE[wps_2](../Token/wps_2_md.md)].

## Tecnologias Relacionadas
O [!INCLUDE[wps_2](../Token/wps_2_md.md)] faz parte de uma família de tecnologias de script relacionadas que ajudarão a automatizar o gerenciamento remoto de computadores baseados no Windows. Incluímos aqui links para obter mais informações sobre essas tecnologias.

-   [Fluxo de trabalho do Windows PowerShell](http://technet.microsoft.com/library/jj134242.aspx). Apresentado pela primeira vez no [!INCLUDE[psversion3](../Token/psversion3_md.md)], o Fluxo de Trabalho do [!INCLUDE[wps_2](../Token/wps_2_md.md)] permite que os profissionais de TI aproveitem os benefícios do [Windows Workflow Foundation](http://msdn.microsoft.com/library/ee342461.aspx) com os recursos de automação e a facilidade de uso do [!INCLUDE[wps_2](../Token/wps_2_md.md)].

-   [Windows PowerShell Web Access](http://technet.microsoft.com/library/hh831611.aspx). O [!INCLUDE[win8_server_2](../Token/win8_server_2_md.md)], apresentado pela primeira vez no [!INCLUDE[wps_2](../Token/wps_2_md.md)] Web Access, atua como gateway do [!INCLUDE[wps_2](../Token/wps_2_md.md)], fornecendo um console do [!INCLUDE[wps_2](../Token/wps_2_md.md)] baseado na Web e direcionado a um computador remoto. Ele permite aos profissionais de TI executar comandos e scripts do [!INCLUDE[wps_2](../Token/wps_2_md.md)] de um console do [!INCLUDE[wps_2](../Token/wps_2_md.md)] em um navegador da Web, sem necessidade de instalar o [!INCLUDE[wps_2](../Token/wps_2_md.md)], software de gerenciamento remoto ou plug-in de navegador no dispositivo cliente.

-   [Serviços Web do Windows PowerShell (Extensão do IIS do Management OData)](http://msdn.microsoft.com/library/windows/desktop/hh880865.aspx). Os Serviços Web do [!INCLUDE[wps_2](../Token/wps_2_md.md)] é uma estrutura para expor facilmente cmdlets do [!INCLUDE[wps_2](../Token/wps_2_md.md)] por meio de um serviço Web baseado em OData executado no servidor Web (IIS).

-   [Introdução à Configuração de Estado Desejado do Windows PowerShell](assetId:///c134aa32-b085-4656-9a89-955d8ff768d0). A Configuração de Estado Desejado (DSC) do [!INCLUDE[wps_2](../Token/wps_2_md.md)], introduzida no [!INCLUDE[psversion4](../Token/psversion4_md.md)], é uma nova plataforma de gerenciamento no [!INCLUDE[wps_2](../Token/wps_2_md.md)] que permite a implantação e o gerenciamento de dados de configuração de serviços de software e o ambiente no qual esses serviços são executados. O DSC fornece um conjunto de extensões de linguagem do [!INCLUDE[wps_2](../Token/wps_2_md.md)], novos cmdlets e recursos que você pode usar para especificar declarativamente como você deseja que o estado do seu ambiente de software seja configurado.

-   O [Windows Management Framework 4.0 Preview](http://go.microsoft.com/fwlink/?LinkID=293881) inclui atualizações do [!INCLUDE[wps_2](../Token/wps_2_md.md)], ISE do [!INCLUDE[wps_2](../Token/wps_2_md.md)], Serviços Web do [!INCLUDE[wps_2](../Token/wps_2_md.md)] (Extensão IIS do Management OData), WinRM (Windows Remote Management), WMI (Windows Management Infrastructure), o provedor do Gerenciador do Servidor WMI e um novo recurso do 4.0, a DSC (Configuração de Estado Desejado) do [!INCLUDE[wps_2](../Token/wps_2_md.md)]. O Windows Management Framework 4.0 Preview permite instalar e usar essas tecnologias em computadores que executam o [!INCLUDE[win8_server_2](../Token/win8_server_2_md.md)], [!INCLUDE[win7_client_firstref](../Token/win7_client_firstref_md.md)] SP1 e [!INCLUDE[win7_server_secondref](../Token/win7_server_secondref_md.md)] SP1.

-   O [Windows Management Framework 3.0](http://www.microsoft.com/download/details.aspx?id=34595) inclui atualizações do [!INCLUDE[wps_2](../Token/wps_2_md.md)], ISE do [!INCLUDE[wps_2](../Token/wps_2_md.md)], Serviços Web do [!INCLUDE[wps_2](../Token/wps_2_md.md)] (Extensão IIS do Management OData), WinRM (Windows Remote Management), WMI (Windows Management Infrastructure) e o provedor do Gerenciador do Servidor WMI. A Visualização do Windows Management Framework 3.0 permite instalar e usar essas tecnologias em computadores que executam o [!INCLUDE[win7_client_firstref](../Token/win7_client_firstref_md.md)] SP1, Windows Server 2008 SP2 e [!INCLUDE[win7_server_secondref](../Token/win7_server_secondref_md.md)] SP1.

## Baixando e atualizando a Ajuda do Windows PowerShell
Os tópicos a seguir descrevem como você pode obter a Ajuda mais recente para [!INCLUDE[wps_2](../Token/wps_2_md.md)] e exibi-lo no prompt de comando do [!INCLUDE[wps_2](../Token/wps_2_md.md)].

-   Cmdlet [Update-Help](http://technet.microsoft.com/library/hh849720.aspx). Um cmdlet do [!INCLUDE[wps_2](../Token/wps_2_md.md)] que baixa e instala as versões mais recentes dos tópicos de ajuda para módulos do [!INCLUDE[wps_2](../Token/wps_2_md.md)] no seu computador.

    Para obter mais informações sobre o sistema de Ajuda Atualizável no [!INCLUDE[wps_2](../Token/wps_2_md.md)], inclusive sobre como instalar a Ajuda Atualizável em computadores isolados da rede, consulte [about_Updatable_Help](http://technet.microsoft.com/library/hh847735.aspx), [Save-Help](http://technet.microsoft.com/library/hh849724.aspx) e [Suporte à Ajuda Atualizável](http://msdn.microsoft.com/library/hh852754.aspx).

-   cmdlet [Get-Help](http://technet.microsoft.com/library/hh849696(v=wps.630).aspx). Um cmdlet do [!INCLUDE[wps_2](../Token/wps_2_md.md)] que você pode usar para aprender sobre os cmdlets e provedores instalados no sistema.

-   Você pode obter notificação sobre as atualizações para os arquivos de Ajuda publicados assinando o seguinte RSS feed: [http://sxp.microsoft.com/feeds/msdntn/PowerShellHelpVersions](http://sxp.microsoft.com/feeds/msdntn/PowerShellHelpVersions).
