# MSFT_DSCLocalConfigurationManager 类的 DisableDebugConfiguration 方法

禁用 DSC 资源调试。

语法
------

```mof
uint32 DisableDebugConfiguration();
```

参数
----------

此方法没有任何参数。

## 返回值
------------

如果成功，则返回零；否则返回错误代码。

## 备注

这是一种静态方法。

## 要求
------------
>**MOF：** DscCore.mof

>**命名空间**：Root\Microsoft\Windows\DesiredStateConfiguration
