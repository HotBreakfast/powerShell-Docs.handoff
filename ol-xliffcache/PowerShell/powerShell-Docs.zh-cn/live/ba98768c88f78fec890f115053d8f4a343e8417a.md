# 适用于 Linux 的内置 Desired State Configuration 资源

资源是可用于编写 PowerShell Desired State Configuration (DSC) 脚本的构建基块。 适用于 Linux 的 DSC 附带了一套用于配置资源（如文件和文件夹、程序包、环境变量、服务和进程）的内置功能。
