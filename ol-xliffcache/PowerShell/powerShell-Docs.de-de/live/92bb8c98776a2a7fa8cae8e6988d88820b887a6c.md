# Zusätzlicher Wert für „RefreshMode“-Eigenschaft

Diese Version bietet den neuen `RefreshMode`-Wert **Disabled**. Wenn dieser Modus festgelegt ist, übernimmt der LCM keine Dokumentverwaltung. Weitere Informationen finden Sie unter [Konfigurieren des lokalen Konfigurations-Managers](../dsc/metaConfig.md).
