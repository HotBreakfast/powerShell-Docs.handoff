# Bewährte Methoden für Pullserver

>Gilt für: Windows PowerShell 4.0, Windows PowerShell 5.0

In den Beispielen in dieser Dokumentation soll erläutert werden, wie PowerShell DSC-Pullserver funktionieren. Die Beispiele sind in den meisten realen praktischen Fällen nicht geeignet. Ausführlichere Informationen zum Einrichten eines Pullservers in einer realen Bereitstellung finden Sie im [Konfigurationsplanungs- und Installationshandbuch](https://github.com/PowerShell/Whitepapers/blob/master/PullServerCPIG/PullServerCPIG.md).<!--HONumber=Feb16_HO4-->
